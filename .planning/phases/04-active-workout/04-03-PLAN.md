  <behavior>
    Previous Performance:
    - Given an exercise_id, returns the most recent completed session's sets for that exercise
    - Caches results in MMKV keyed by exercise_id to avoid Supabase round-trips
    - Returns null if no previous session exists
    - Cache is populated after each session completion

    PR Detection:
    - checkForPR(exerciseId, loggedWeight, prBaselines, tracksPR) returns { isPR, previousBest }
    - If exercise does not track PRs, returns { isPR: false, previousBest: null }
    - If no baseline exists and exercise tracks PRs, returns { isPR: true, previousBest: null } (first-time baseline)
    - If loggedWeight > baseline.weight, returns { isPR: true, previousBest: baseline.weight }
    - If loggedWeight <= baseline.weight, returns { isPR: false, previousBest: baseline.weight }
    - Session-local PR cache: if a PR was already detected this session for the same exercise, subsequent sets compare against the session PR (not stale baseline)
  </behavior>
  <action>
    1. Create `src/features/workout/hooks/usePreviousPerformance.ts`:
       - Uses a named MMKV instance `previous-performance-cache`
       - `getPreviousPerformance(exerciseId)`: reads from MMKV cache first. If cache miss, queries Supabase: `workout_sessions` joined with `session_exercises` and `set_logs` where exercise_id matches, ordered by ended_at DESC, limit 1. Caches result in MMKV.
       - `cachePreviousPerformance(session)`: after session completion, store each exercise's logged sets in cache keyed by exercise_id. Called from useWorkoutSession.finishWorkout().
       - Returns: { previousSets, isLoading } for a given exercise_id

    2. Create `src/features/workout/hooks/usePRDetection.ts`:
       - Export pure function `checkForPR(exerciseId, loggedWeight, prBaselines, tracksPR)` as described in RESEARCH.md Pattern 5
       - Export hook `usePRDetection()` that:
         - Loads PR baselines from Supabase on mount (cache in component state)
         - Loads exercise track_prs flags from exerciseStore
         - Maintains a `sessionPRCache: Map<string, number>` — updated when a PR is detected in this session
         - `detectPR(exerciseId, weight)`: calls checkForPR, if PR found updates sessionPRCache and enqueues Supabase upsert to pr_baselines (exercise_id + weight), returns { isPR, previousBest }
         - Returns: { detectPR, sessionPRCache }

    3. Create `tests/workout/pr-detection.test.ts`:
       - Test: non-tracked exercise returns isPR false
       - Test: tracked exercise with no baseline returns isPR true (first-time)
       - Test: tracked exercise with weight above baseline returns isPR true
       - Test: tracked exercise with weight equal to baseline returns isPR false
       - Test: tracked exercise with weight below baseline returns isPR false
       - Test: session PR cache — second set at higher weight than first PR in same session still detects PR

    4. Create `tests/workout/previous-performance.test.ts`:
       - Test: returns null when no previous session exists
       - Test: returns previous sets when cache hit
       - Test: getPreviousPerformance returns correct shape (exercise_id, sets array, session_date)
  </action>
  <verify>
    <automated>npx jest tests/workout/pr-detection.test.ts tests/workout/previous-performance.test.ts --bail</automated>
  </verify>
  <done>checkForPR is a tested pure function with all edge cases covered. usePreviousPerformance caches in MMKV. usePRDetection maintains session-local PR cache to prevent race conditions. All tests pass.</done>

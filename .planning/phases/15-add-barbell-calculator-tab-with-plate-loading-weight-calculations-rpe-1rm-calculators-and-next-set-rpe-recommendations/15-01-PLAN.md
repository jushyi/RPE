**1. Create `src/features/calculator/types.ts`:**
```typescript
export interface PlateBreakdown {
  plates: { weight: number; count: number }[];
  remainder: number;
}

export interface BarPreset {
  label: string;
  weightLb: number;
  weightKg: number;
}

export interface NextSetInput {
  lastWeight: number;
  lastReps: number;
  lastRpe: number;
  targetRpe: number;
  targetReps: number;
  unit: 'kg' | 'lbs';
}

export interface NextSetResult {
  recommendedWeight: number;
  percentChange: number;
  explanation: string;
}
```

- `LB_PLATES = [45, 35, 25, 10, 5, 2.5]` (descending for greedy algo)

**3. Create `src/features/calculator/utils/plateCalculator.ts`:**
- Export `calculatePlates(targetWeight, barWeight, availablePlates): PlateBreakdown`
- If targetWeight <= barWeight, return `{ plates: [], remainder: 0 }`
- Calculate perSide = (targetWeight - barWeight) / 2
- Greedy algorithm: iterate availablePlates (already descending), floor(perSide / plate) for count
- Use `Math.round(x * 100) / 100` after each subtraction to avoid floating point drift
- Return remainder = whatever is left after all plates assigned

- Export `getWeightForRpeAndReps(e1rm, rpe, reps)`: lookup percentage, return `Math.round((e1rm * pct) / 100 * 10) / 10`

**2. Create `src/features/calculator/utils/nextSetCalc.ts`:**
- Import `RPE_TABLE` from `./rpeTable` and types from `../types`
- Export `calculateNextSet(input: NextSetInput): NextSetResult`
  - Derive e1RM from last set: `lastWeight / (lastPct / 100)` where lastPct comes from RPE_TABLE[lastRpe][lastReps-1]
  - Derive target weight: `e1RM * (targetPct / 100)` where targetPct from RPE_TABLE[targetRpe][targetReps-1]
  - Round to loadable: `roundToLoadable(rawWeight, unit)`
  - Calculate percentChange: `((recommended - lastWeight) / lastWeight) * 100`, rounded to 1 decimal
  - Generate explanation string: `"Based on e1RM of {e1rm} {unit}, {targetReps} reps at RPE {targetRpe} = {targetPct}% of 1RM"`
  - Handle invalid RPE/reps gracefully: return lastWeight, 0% change, error explanation
- Export `roundToLoadable(weight, unit)`: round to nearest 5 (lbs) or 2.5 (kg)

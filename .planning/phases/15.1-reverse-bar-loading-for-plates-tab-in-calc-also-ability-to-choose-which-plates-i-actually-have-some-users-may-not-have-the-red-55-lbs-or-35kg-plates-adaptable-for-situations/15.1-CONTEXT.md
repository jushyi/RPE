# Phase 15.1: Reverse Bar Loading + Customizable Plate Inventory - Context

**Gathered:** 2026-03-13
**Status:** Ready for planning

<domain>
## Phase Boundary

Enhance the Plates sub-tool in the Calculator tab with two additions: (1) a reverse calculator mode where users tap plates to build up total weight (opposite of the existing weight-to-plates mode), and (2) customizable plate inventory so users can disable plate sizes they don't own, affecting calculations in both modes.

</domain>

<decisions>
## Implementation Decisions

### Reverse calculator mode
- Toggle within the Plates tab: segmented control at top switching between "Weight > Plates" and "Plates > Weight" modes
- Both modes share the same barbell diagram, bar preset picker, and unit toggle
- Reverse mode shows +/- steppers per plate size (one row per enabled plate size)
- Counts are per-side; total weight displayed prominently above the diagram
- Barbell diagram updates live as plates are added/removed via steppers
- "Clear All" button to reset to empty bar
- No artificial limit on plate count per size — unlimited
- Disabled plates (from inventory) are hidden from the stepper list, not grayed out

### Plate inventory (My Plates)
- Inline on the Plates tab as a collapsible "My Plates" section
- Collapsed by default once configured; shows summary like "5 of 7 plates enabled"
- Tap to expand and toggle individual plate sizes on/off
- Separate inventories for lb and kg (independent toggle states)
- Affects both calculator modes: weight-to-plates only uses enabled plates; reverse mode only shows enabled plates as steppers
- Default: all plates enabled for new users

### Persistence
- MMKV local storage only (no Supabase sync) — plate availability is gym-specific
- Each unit system (lb/kg) has its own persistent inventory in MMKV
- Switching units loads that unit's previously configured inventory

### Edge cases
- When inventory can't reach target weight exactly: show closest loadable weight + remainder warning
- Remainder warning mentions the missing plate: "Cannot load exactly — 2.5 lb plates not in your inventory"
- Reverse mode unlimited plate counts — diagram expands horizontally as needed

### Claude's Discretion
- Exact stepper UI styling (+/- buttons, count display)
- Collapsible section animation
- "My Plates" section placement relative to other controls
- Clear All button placement and styling
- Segmented control labels for the two modes

</decisions>

<specifics>
## Specific Ideas

- Reverse mode is a "plates > weight" calculator — user builds up a barbell by selecting plates and sees the total
- +/- steppers per plate size feels more precise than tap-to-add for gym use
- Collapsible "My Plates" section keeps the UI clean once configured — most users set it once and forget
- Missing plate warning is actionable: tells user which plate they need, not just that there's a gap

</specifics>
